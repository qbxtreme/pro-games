# Meme Car

**Hub ID:** `meme-car`  
**Path:** `games/meme-car-race/`  
**Play:** [http://localhost:8080/games/meme-car-race/](http://localhost:8080/games/meme-car-race/)

## Overview

Side-scrolling meme car racer inspired by All Out’s Meme Car. Collect meme-themed cars, race solo for distance, or PVP with blasters on shared tracks.

## Save & multiplayer

- **Save key:** `memeCarRaceSave`
- **MP room:** `meme-car-race`

## Worlds

| World | Theme | Jump difficulty |
|-------|-------|-----------------|
| Oasis | Desert | Easy |
| Steampunk | Gears/pipes | Hard |
| Cyber | Neon city | Hardest |

## Meme cars

Rizz Rod (free) → Skibidi Mobile → Sigma Drift → … → Grimace Shake (50k). Each has price, speed multiplier, and meme skin.

## Modes

- **Solo:** 3 lives; dodge obstacles; distance = score
- **PVP:** 2 HP; fire blasters; rank by distance; tracks `pvpWins`
