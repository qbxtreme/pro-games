# Mob Battle

**Hub ID:** `mob-battle`  
**Path:** `games/mob-battle/`  
**Play:** [http://localhost:8080/games/mob-battle/](http://localhost:8080/games/mob-battle/)

## Overview

Capture-and-battle RPG using the same core loop as Dino Park / capture templates but with Mob Battle branding. Walk the world, battle wild mobs, capture creatures, and level up your trainer.

## Save & multiplayer

- **Save key:** `mob-battle`
- **MP room:** `mob-battle`

## Mechanics

- Joystick + WASD movement
- Tranq/capture wild mobs on win
- Feed, level, and build your squad
- Zone navigation bar at bottom
- Turn-based battles with flee option

## Key files

| File | Role |
